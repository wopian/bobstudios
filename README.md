### Setup
```bash
# Check you're using NodeJS 7.0.0 or above
node -v
# Check you're using npm 4.0.0 or above
npm -v
# Download source code
git clone https://github.com/wopian/bobstudios.git
cd bobstudios
# Install dependencies
npm i
```
### Building
```bash
gulp
```
### Updating
```bash
npm update && npm prune
```
